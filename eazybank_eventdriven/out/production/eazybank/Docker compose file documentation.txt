Docker compose commands:
	
-You can create on comman configuration file in any microservice to run all docker container instead of doing run with separate commands
	
-Click on any microservice that you want to create docker compose file 
	- right click create file name with "Docker-compose.yml"
				services:
				 accounts:
				  image: "nins07/accounts:eazyaccountsv1"
				  container_name: accounts-ms
				  ports:
			      - "8080:8080"
			     deploy:
				  resources:
				   limits:
				    memory: 700m
			     networks:
				  - eazybank
		        loans:
			     image: "nins07/loans:eazyloansv1"
			     container_name: loans-ms
			     ports:
			      - "8090:8090"
			    deploy:
			     resources:
				  limits:
				   memory: 700m
			     networks:
			      - eazybank

		        card:
			     image: "nins07/card:eazycardsv1"
			     container_name: cards-ms
			     ports:
			      - "9000:9000"
			    deploy:
			     resources:
				  limits:
				   memory: 700m
			    networks:
			     - eazybank

		       networks:
		        eazybank:
		         driver: "bridge"
				 
	-Go to that perticular microservices file open cmd prompt
	-Run the command "docker compose up -d"
	-To stop running container "docker compose stop"
	-TO stop and delete running container "docker compose down"
